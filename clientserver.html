    WEB BASIC: client&server

    1. internet VS web
    2. web browser&web server
    3. web server&http

      4. internet VS web

      internet이 도로라면 web은 도로 위 자동차 한대
      internet이 도시라면 web은 건물 하나
      web은 internet의 많은 기능 중 하나

      web browser&web server

      web browser의 request를 받아서 web server에서 response
      -> client 와 server의 관계
      (web browser는 웹을 요청하고 web server는 반환)
      +web hosting은 이 서버 관리를 대신 해주는 것

      web server&http

      web의 주소가 다음과 같다고 하자
      http://localhost/index.html
      여기서 localhost와 같이 의미가 있는 이름을 domain name이라 하며 자신의 컴퓨터를 가리키는 것
      -> IP주소로 표현 하자면 127.0.0.1 (internet protocol address)

      http -> hypter text transfer protocol
      -> web을 server와 client 사이에서 전송하기 위한 통신 규약
      -> file:// 로 시작하지 않고 http://로 시작한다면 http 통신 규약에 따라 server 접속

      web client에서 server에게 web을 요청할 때 그 server의 IP주소가 필요
      -> 네트워크 설정-IPv4 주소 가 바로 자신의 IP주소
